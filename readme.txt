=== WooCommerce Affirm Gateway ===
Author: WooCommerce
Tags: woocommerce
Requires at least: 4.4
Tested up to: 4.8
Requires WooCommerce at least: 2.6
Tested WooCommerce up to: 3.2

Accept installment payments in your WooCommerce store via Affirm

See http://docs.woocommerce.com/document/woocommerce-affirm-gateway/ for full documentation.

== Installation ==

1. Upload the entire 'woocommerce-gateway-affirm' folder to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
